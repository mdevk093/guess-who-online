# Guess Who Online - Client

The frontend application for "Guess Who Online", built with React and Vite.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

## Running the Client

### Development Mode

```bash
npm run dev
```
The application will be available at [http://localhost:5173](http://localhost:5173).

### Building for Production

```bash
npm run build
```
The build artifacts will be located in the `dist/` directory.

## Features

- Real-time connection to the game server.
- Dynamic character board.
- Responsive design.
